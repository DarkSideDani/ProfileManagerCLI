# Profile Manager CLI

A C++ command-line application for creating and managing user profiles.

## Features (WIP)
A command-line application that lets you:
1. Create a profile
2. View a profile
3. List all profiles
4. Update a profile
5. Delete a profile
6. Add/remove hobbies
7. Save/load profiles from disk (JSON-like or simple text format)

## Tech
- C++17
- CMake
- STL containers
